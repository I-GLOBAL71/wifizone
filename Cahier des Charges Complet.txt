## Système de gestion WiFi avec paiement mobile (Campay)

## 🎨 IDENTITÉ VISUELLE & PHILOSOPHIE SPIRITUELLE

### Palette de couleurs sacrée
**Symbolisme pour l'ascension et la prospérité**

- **Violet profond (#6B46C1 - #7C3AED)** : Spiritualité, sagesse, transformation
- **Or lumineux (#F59E0B - #FBBF24)** : Abondance, succès, énergie divine
- **Bleu nuit (#1E3A8A - #3B82F6)** : Confiance, profondeur, stabilité
- **Vert émeraude (#059669 - #10B981)** : Croissance, harmonie, prospérité
- **Blanc pur (#FFFFFF)** : Clarté, pureté, nouveaux départs
- **Gris lunaire (#F3F4F6 - #E5E7EB)** : Neutralité, équilibre

**Gradient principal** : Violet → Or (symbolise le passage de la sagesse à l'abondance)

### Principes spirituels du design
1. **Fluidité** : Transitions douces représentant l'eau qui s'écoule (prospérité)
2. **Lumière** : Effets lumineux subtils évoquant l'illumination
3. **Cercles** : Géométrie circulaire (cycles, complétude, infini)
4. **Nombre d'or** : Proportions harmonieuses (ratio 1.618)
5. **Respiration** : Espaces blancs généreux (clarté mentale)

## 📱 INTERFACE UTILISATEUR (User App)

### Objectif
Permettre à **toute personne** (même sans culture digitale) d'acheter du forfait WiFi en **moins de 3 clics**.

#### Caractéristiques
- **Animation d'entrée** : Logo qui pulse doucement (battement de cœur)
- **Fond** : Gradient violet-or avec effet de particules lumineuses animées

- Mode nuit automatique selon l'heure

## 📐 SPÉCIFICATIONS TECHNIQUES UI/UX

### Performance

**Objectifs**
- **First Contentful Paint** : < 1.5s
- **Time to Interactive** : < 3s
- **Lighthouse Score** : > 90

**Optimisations**
- Images WebP avec fallback
- Lazy loading images/components
- Code splitting (React.lazy)
- Service Worker pour cache offline
- CDN pour assets statiques

---

### Accessibilité (WCAG 2.1 AA minimum)

**Contraste**
- Texte corps : 4.5:1 minimum
- Texte large (18px+) : 3:1 minimum
- UI interactive : 3:1 minimum

**Navigation clavier**
- Tab order logique
- Focus visible (outline 2px violet)
- Raccourcis clavier (admin) : Ctrl+S (save), Escape (fermer modale)

**Screen readers**
- ARIA labels sur tous les éléments interactifs
- Alt text descriptifs pour images
- Landmarks (header, nav, main, footer)
- Live regions pour notifications

**Motion**
- Respecter `prefers-reduced-motion`
- Option "Réduire les animations" dans settings

---

### Internationalisation (i18n)

**Langues prioritaires**
1. Français (default)
2. Anglais

**Système**
- Fichiers JSON séparés (/locales/fr.json, /locales/en.json)
- Détection auto langue navigateur
- Sélecteur de langue visible (flag icons)
- Format dates/heures localisé
- Devise adaptée (FCFA vs XAF)

---

### Sécurité Frontend

**Bonnes pratiques**
- Pas de clés API dans le code (env vars)
- Validation côté client + serveur
- Sanitization inputs (XSS prevention)
- HTTPS obligatoire
- CSP (Content Security Policy) headers
- SameSite cookies

**Gestion tokens**
- JWT stocké en httpOnly cookie (si possible)
- Refresh token automatique
- Logout sur expired token
